# My_pet_diary
---

## 概要

#ペットの健康管理、気象データの連動、健康異常を検知、日々の健康記録日記アプリです。

---

## 🐾 主な機能
* **ペットの健康日記**: 毎日の食事、排泄、活動量を記録
* **気象データ連動**: 気圧や気温の変化とペットの体調相関を可視化
* **異常検知システム**: 普段のデータから体調不良の兆候を早期発見
