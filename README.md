# My_pet_diary
---

## 概要

ペットの健康管理、気象データの連動、健康異常を検知、日々の健康記録日記アプリです。

---

## 仕様書
- URL：https://www.notion.so/MyPetDiary-344babb785ac8062af3cc5ba1ad5499d?source=copy_link

---

### 🐾 主な機能
* **ペットの健康日記**: 毎日の食事、排泄、活動量を記録
* **気象データ連動**: 気圧や気温や天気によって今日のペットの日常に変化が出ないかの判断材料になる

---

#### 🚀 今後のアップデート予定
🚀 今後のアップデート予定 (Roadmap)
MyPetDiary は、より楽しく、より繋がりを感じられるアプリを目指して、以下の機能を順次追加予定です。

- 📹 メディア機能の拡充
動画投稿機能
静止画だけでなく、ペットの愛くるしい動きをそのまま記録できるよう、mp4形式などの動画アップロードに対応します。

- 🤝 コミュニケーション機能
ユーザー間での日記交換
特定のフレンドと日記を見せ合い、アドバイスや励ましを送れる交換日記機能。

- SNSライクなタイムライン投稿
自分の日記を「公開投稿」としてシェアし、他の飼い主さんと繋がれるオープンなプラットフォーム化。

- 💬 インタラクションの強化
投稿への返信（リプライ）機能
他のユーザーの投稿に対して、コメントで直接コミュニケーションを取れるようにします。

- 「いいね！」リアクション機能
素敵な写真や役立つ記録に対して、ワンタップで共感を示せるリアクションボタンを実装します。

- アカウント消去機能
  登録機能がある一方で消去機能がない問題点があるため、今後、実装します
